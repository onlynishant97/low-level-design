# ATM System — High Level Design (HLD) Challenge

## 📋 Problem Statement

Design a **High Level System Design** for an ATM (Automated Teller Machine) network
that supports multiple ATMs connected to a centralized banking backend.
The system should be scalable, highly available, and secure.

---

## 🎯 Functional Requirements

- Users should be able to **authenticate** themselves using a card and a PIN.
- Users should be able to perform a **balance inquiry** on their account.
- Users should be able to **withdraw cash** from their account.
- Users should be able to **deposit cash** into their account.
- Users should be able to view their **transaction history**.
- The system should **communicate with the bank's backend** to validate accounts
  and perform transactions.
- The ATM should **dispense cash** based on available denominations.
